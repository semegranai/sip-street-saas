# Sip Steet – Internal Development Modeling Platform

**Sip Steet Group** is an active, vertically integrated real estate development company specializing in SIP (Structural Insulated Panels) technology. We develop residential (luxury to workforce housing), commercial, industrial, hospitality, data centers, and other asset classes.

We are rebuilding our entire internal technology stack. The goal is to replace fragmented Excel templates with a single, powerful, consistent web-based modeling platform.

### Current Project Scope (Phase 1)
- Web-based financial modeling tool
- Zillow/MLS link dropping with auto-population
- Editable sub-component cost breakdowns
- Unified P&L, GP/LP waterfall, cash waterfall
- Professional, standardized PDF report generation
- Clean 2-tab interface (Assumptions + Dashboard)

### Technology Goals
- High accuracy and consistency across all calculations
- Excellent usability for internal deal teams
- Scalable architecture for future multi-asset class expansion
- Strong foundation for AI-assisted features

Live demo: https://sip-street-saas.vercel.app/

See `ROADMAP.md`, `FEATURES.md`, and `TECHNICAL-SPECS.md` for full details.
